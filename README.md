# Portfolio

> 業務課題の整理から設計・実装までを一貫して対応  
> Google Apps Script（GAS）を中心とした個人・小規模事業者向け業務効率化システム

---

## 📖 このポートフォリオについて

実際の業務ヒアリングをもとに、**「導入しやすさ」「運用のしやすさ」**を重視して設計・実装したシステム集です。

各プロジェクトでは、実装コードだけでなく、  
**「どのような課題を想定し、どのような運用を目指したか」**を重視してまとめています。

実際の利用シーンや業務フローを想定し、  
**「作って終わり」ではなく、運用しながら育てていくシステム**を目指しています。

---

## 🎯 開発スタンス

**「技術で課題を解決し、日々の業務を楽にする」**

技術ありきではなく、  
**「何が一番困っているのか」「どう運用されるのか」**を起点に設計することを大切にしています。

業務課題のヒアリングから設計・実装までを重視し、  
**個人・小規模事業者向けの業務効率化システム**を中心に開発しています。

### 1. 顧客管理＆請求書発行システム（GAS）🏢

> 個人・小規模事業者向けに、顧客情報管理から請求書発行・保存までを一元化した業務基盤システム

**課題解決のポイント**
- 顧客情報と請求書を紐づけて管理し、検索性・正確性を向上
- 請求書発行やファイル整理の手作業を削減（作業時間70〜80%削減）
- スプレッドシートより見やすく、使いやすいUI設計

**主な機能**
- 顧客情報の登録・検索・管理
- 請求書の自動作成・PDF化・フォルダ保存
- Gmail下書き自動作成（任意）
- 支払状況管理

**技術スタック**  
Google Apps Script / JavaScript / HTML / CSS

[📄 README・画面構成・運用イメージを見る](https://github.com/usako-ui/customer-management-invoice-system-portfolio)

### 2. Googleカレンダー自動日程調整（GAS）📅

> 個人事業主の実運用をヒアリングし、日程調整の往復連絡を不要にした予約自動化システム

**課題解決のポイント**
- 日程調整の往復連絡を削減し、URLひとつで予約完結
- 手動調整・やり取りの手間を削減

**主な機能**
- Googleカレンダーと連携した空き時間の自動表示
- 予約確定時の自動通知
- 予約の重複防止（排他制御）
- リマインダー機能

**技術スタック**  
Google Apps Script / Google Calendar API

**詳細はこちら**  
[📄 README・画面設計・システム構成を見る](https://github.com/usako-ui/GAS-google-calendar-auto-reservation)
