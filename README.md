# Portfolio

業務課題の整理から設計までを重視し、  
実際の業務ヒアリングをもとに、Google Apps Script（GAS）を中心とした  
個人・小規模事業者向けの業務効率化システムを設計・実装してきたポートフォリオ集です。

各システムは、実際の利用シーンや業務フローを想定し、  
「導入しやすさ」「運用のしやすさ」を重視して設計しています。

本ポートフォリオでは、実装コードだけでなく、  
「どのような課題を想定し、どのような運用を目指したか」を重視してまとめています。

各プロジェクトの詳細は、リンク先の README・画面構成をご覧ください。
---

## ポートフォリオ一覧

### Googleカレンダー自動日程調整（GAS）📅
>　個人事業主の実運用をヒアリングし、日程調整の往復連絡を不要にした予約自動化システム。

- 概要：Googleカレンダーと連携し、URLひとつで日程調整を完結させる予約管理ツール
- ポイント：手動調整・やり取りの手間を減らし、実運用を想定した排他制御・通知設計
- 内容：README/画面設計・システム構成・スクリーンショット・運用イメージ
- 技術：Google Apps Script
- 詳細はこちら：
 [https://github.com/usako-ui/GAS-google-calendar-auto-reservation]

### 顧客管理＆請求書発行システム（GAS）🏢
>　個人・小規模事業者向けに、顧客情報管理から請求書発行・保存までを一元化した業務基盤システム。

- 概要：顧客情報の登録・検索から、請求書の作成・保存・管理までを一元化した、個人・小規模事業者向け業務管理システム
- ポイント：顧客情報と請求書を紐づけて管理し、検索性・正確性を高めつつ、請求書発行やファイル整理の手作業を削減
- 内容：README/画面設計・システム構成・スクリーンショット・運用イメージ
- 技術：Google Apps Script
- 詳細はこちら：
 [https://github.com/usako-ui/customer-management-invoice-system-portfolio]

### くらしログ（GAS）🏠
>  日常の記録を無理なく継続できるよう設計した、個人向けの生活ログ管理システム。

- 概要：家庭内の支出・在庫・生活メモを一元管理するツール
- ポイント：入力負荷を抑え、振り返りや改善につなげる設計
- 内容：README/画面設計・スクリーンショット・運用イメージ
- 技術：Google Apps Script
- 詳細はこちら：
  [https://github.com/usako-ui/kurashi-log-portfolio]

### 小規模飲食店向け 在庫管理システム（React + Supabase）🍔
>  現場の業務フローを想定しながら、在庫・仕入・売上・廃棄を一元管理するシステム。

- 概要：個人事業主向け在庫管理Webアプリ
- ポイント：小規模飲食店における食材在庫の把握・管理を目的とした  業務支援用の在庫管理システム
- 内容：README/画面設計・スクリーンショット・運用フロー
- 技術：React / Supabase
- 詳細はこちら：
  [https://github.com/usako-ui/inventory-management-portfolio]
